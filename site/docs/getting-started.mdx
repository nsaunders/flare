# Getting Started

## Browser

Each live demo featured on the [Examples](/examples) page can be used as the
starting point for a [CodeSandbox](https://codesandbox.io/) project, allowing
you to experiment with Flare directly in the web browser. Click on the pencil
icon next to a demo to get started.
